Overview
------------------

This module interfaces to MAX3421 based USB Peripheral and Host Controller.


Wiring
-------------------

| MAX3421   | Espruino   |
|-----------|------------|
| GND       | GND        |
| 3.3V      | 3.3V       |
| MOSI      | MOSI (B3)  |
| MISO      | MISO (B4)  |
| SCK       | SCK  (B3)  |       
| SS        | A0         |
| MAX RESET | A1 (3.3V)  |

Usage
-------------------

Usage is very simple. Using the example pins shown above:

```JavaScript 
SPI1.setup({sck: B3, miso: B4, mosi: B5, mode: 0); // mode 3 should also work  
var max3421=require("MAX3421");
```
